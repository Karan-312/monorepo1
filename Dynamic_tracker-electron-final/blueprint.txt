DSA Revision Tracker - Project Blueprint
========================================

1. UI Foundation
----------------
- Modern layout: header, main content area
- Table/grid to display DSA problems and revision data

2. Data Model & State Management
-------------------------------
- Problem structure: name, attempts (dates, status, time), next review, last performance
- Use React state (or Zustand/Redux if needed)

3. Basic CRUD Functionality
--------------------------
- Add new problem
- Edit problem details
- Delete problem

4. Revision Logic
-----------------
- Scoring system: +2 for "Myself", +1 for "Help"
- Calculate performance rating (Excellent, Good, Mid, Poor, Skipped)
- Auto-calculate next review date
- Update last performance

5. Persistence
--------------
- Start with localStorage for data
- (Optional) Use Electron file system APIs for robust saving/loading

6. UI/UX Enhancements
---------------------
- Keyboard shortcuts
- Quick add
- Smooth navigation
- Modern styling (Material UI, Tailwind, or custom CSS)

7. Notifications (Optional)
--------------------------
- Electron notification API for review reminders

8. Packaging
------------
- Build and package app for Windows (and optionally other platforms)
